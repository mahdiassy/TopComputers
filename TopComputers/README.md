# TopComputers - Complete E-commerce Solution

A modern, responsive e-commerce platform built with React, TypeScript, and Firebase. Features both a public storefront and admin panel for comprehensive store management.

## üöÄ Features

### Public Storefront
- **Modern Design**: Responsive UI with dark/light mode
- **Product Catalog**: Advanced filtering and search capabilities
- **Product Details**: Image galleries, specifications, reviews
- **Secret Admin Access**: Hidden gesture-based navigation to admin panel

### Admin Panel
- **Dashboard**: Overview of store metrics and recent activity
- **Product Management**: Full CRUD operations for products
- **Category Management**: Organize products into categories
- **Brand Management**: Manage product brands
- **Inventory Tracking**: Stock levels and low stock alerts
- **Image Upload**: Integrated file upload for product images
- **Responsive Admin UI**: Works on desktop, tablet, and mobile
